/model:
    Contém todas as classes de modelo. Local onde ficam as regras de negocio e a conexão com o banco.

/view:
    Contém todas as classes visualização. Cria as janelas por onde o usuario interage com o sistema.

/controller:
    Contém todas as classes de controle. Fazem a ligação da camada View com o Model.

/utils:
    Contém classes de funções e classes uteis para o sistema, como, classe de conexão do banco e a classe de
    janelas do programa.
